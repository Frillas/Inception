Projet Inception

Infrastructure Docker multi-services

aroullea – 42 Angoulême

À propos

Le projet Inception de l'école 42 est une initiation à l'administration système et aux conteneurs Docker. Ce projet vise à virtualiser plusieurs images Docker en créant des conteneurs dans une machine virtuelle personnelle.

Services

  • Nginx – Serveur web HTTPS
  • WordPress – Système de gestion de contenu
  • MariaDB – Base de données
  • Redis – Cache
  • FTP – Transfert de fichiers

Accès

  • Admin WordPress : https://aroullea.42.fr/wp-admin
